# itüöder website

Astro tabanli itüöder websitesi.

## Cloudflare kurulum notlari

Bu repo Cloudflare Pages / Workers import akisi icin hazirlandi.

- Framework: Astro
- Build command: `npm run build`
- Build output directory: `dist`
- Node version: `22`

Cloudflare panelinden `baranuyukus/ituoder-website` reposunu import ederek otomatik deploy baslatabilirsin.
